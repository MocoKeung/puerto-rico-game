## Generation Notes

All assets should be generated as SVG files for scalability and small file size. Colors should follow the PR board game scheme:
- **Green:** Government buildings, special tiles
- **Purple:** Production buildings, markets
- **Brown:** Wooden elements, ships
- **Yellow/Gold:** Resources, victory points
- **Blue/Gray:** UI elements, backgrounds

Ready for SVG generation pipeline implementation.
